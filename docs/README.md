## Quick Start

```javascript
import { SemanticKernel, SymbolicAI } from '@aleph-ai/tinyaleph';

// Create a semantic kernel
const kernel = new SemanticKernel();

// Encode concepts as prime signatures
const wisdom = kernel.encode('wisdom');
const knowledge = kernel.encode('knowledge');

// Calculate semantic similarity
const similarity = kernel.similarity(wisdom, knowledge);
```

## Core Concepts

### Prime Semantics
Every concept maps to a unique prime factorization, creating an arithmetic space where:
- **Similarity** = shared prime factors
- **Composition** = multiplication of signatures
- **Interference** = phase alignment of wave representations

### High-Dimensional Algebra
The library uses hypercomplex numbers (quaternions → octonions → sedenions) to represent:
- 4D: Spatial rotations and orientations
- 8D: Semantic relationships with non-associative structure
- 16D: Full cognitive state representation

### Resonance Networks
Concepts form networks where:
- Nodes are prime-encoded meanings
- Edges are resonance strengths (phase coherence)
- Dynamics follow Kuramoto synchronization
